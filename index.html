Resume

TrailMix (AllTrails Clone)

TrailMix is an AllTrails clone where users are able to search for their favorite trails or parks. Once on a park page a user is provided with more in-depth information about the park, a map of the park, directions to the park, and popular trails within the park. On the trail page a user is provided with information similair to the park page, as well as the ability to leave a review of the trail. TrailMix backend is built using Ruby on Rails framework and PostgreSql database. The frontend of the site is built with React, using Redux to manage the frontend state. Jquery $.ajax() function is used for all the AJAX.

Not Sorry (multiplayer sorry game)

Not Sorry is an online multiplayer game where users can play solo or with friends. Users can create a game lobby for players to join. Once the desired number of players has join the created of the lobby can start the game. Once the game starts a board will appear and the player who gets their piece around the game board and through the safezone is the winner! Using socket.io players can experience real time gameplay. Useing a Node.js/ MongoDb to manage the backend and react/ express to manage the frontend.

Solar System Simulation

In this projects users will be given a simulation of our planets solar system rotation around the sun. On Click users will be provided with in-depth information about the planet they have clicked on. Such as name, surface temperature, and oribtal period. Implemented canvas to simulate the rotation of the planets.
